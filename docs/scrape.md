## 四、爬虫配套过滤规则（脚本内置筛选逻辑，减少无效图）
1. 优先筛选：黑白线稿、连环画内页、原稿扫描、300dpi高清、无现代滤镜
2. 过滤关键词（匹配到直接丢弃图片）：现代插画、AI绘图、彩色重绘、海报、油画、短视频截图、书法单字、人物照片
3. 存储目录规则（自动分层，方便AI训练读取）
```
comic_reference/
├─刘继卣/
│  ├─刘继卣_武松打虎_01.jpg
│  ├─刘继卣_大闹天宫_02.jpg
├─刘锡永/
├─陈光镒/
...
```

## 五、爬虫搜索站点优先级（按图片清晰度、版权友好排序）
1. 中国连环画数字图书馆 lib.sinocomic.com（馆藏原版，无水印）
2. 连艺网 lhhart.com（画家专题高清原稿）
3. 百度图片（通用兜底，使用general_search_word）
4. 孔夫子旧书网图集、豆瓣连环画小组高清扫描帖

import os

# 读取上方json文本
with open("comic_artist_search.json", "r", encoding="utf-8") as f:
    data = json.load(f)

for item in data:
    artist = item["artist_name"]
    save_names = item["image_names"]
    search_words = item["search_keywords_general"]
    # 创建画家文件夹
    os.makedirs(f"./comic_reference/{artist}", exist_ok=True)
    # 循环每个关键词爬取，按save_names依次命名保存
    for idx, kw in enumerate(search_words):
        target_name = save_names[idx]
        print(f"正在爬取：{kw}，保存文件名：{target_name}")
